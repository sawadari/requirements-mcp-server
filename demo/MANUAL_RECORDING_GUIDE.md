# 手動録画ガイド

デモ動画を手動で録画する際の手順書です。音声ファイルは既に生成済みなので、それをガイドに操作を実演してください。

## 準備

1. **OBS Studio を起動**
   - 録画設定を確認（解像度、フレームレート）
   - 録画先フォルダを確認

2. **音声ファイルを確認**
   - 場所: `demo/narrations/01-basic-usage/`
   - Windows Media Player や VLC などで再生できることを確認

3. **VSCode を起動**
   - ターミナルを2つ開く準備
   - Claude Code が使える状態にする

4. **画面レイアウト**
   - VSCode を中央に配置
   - 音声プレーヤーは画面外に配置（録画されない場所）

---

## 録画手順

### 📍 シーン1: イントロダクション（5秒）

**音声**: `intro.wav` を再生
> "requirements-mcp-serverは、Claude Codeと連携する要求管理システムです。"

**操作**:
- 特になし（VSCode の画面を表示するだけ）

**待機時間**: 5秒

---

### 📍 シーン2: MCPサーバー起動（15秒）

**音声**: `start-mcp-server.wav` を再生
> "まず、MCPサーバーを起動します。ターミナルでnpm run devコマンドを実行すると、21個のMCPツールが読み込まれます。"

**操作**:
1. `Ctrl + \`` でターミナルを開く
2. 以下のコマンドを入力（またはコピペ）:
   ```bash
   cd requirements-mcp-server
   npm run dev
   ```
3. サーバーが起動し、ツールがロードされるのを待つ（約8秒）

**コピペ用コマンド**:
```
cd requirements-mcp-server
npm run dev
```

**待機時間**: 15秒

---

### 📍 シーン3: Webビューアー起動（12秒）

**音声**: `start-web-viewer.wav` を再生
> "次に、Webビューアーを起動します。別のターミナルでnpm run view-serverを実行します。"

**操作**:
1. `Ctrl + Shift + \`` で新しいターミナルを開く
2. 以下のコマンドを入力:
   ```bash
   npm run view-server
   ```
3. ビューアーが起動するのを待つ（約5秒）

**コピペ用コマンド**:
```
npm run view-server
```

**待機時間**: 12秒

---

### 📍 シーン4: Claude Codeで要求追加（20秒）

**音声**: `add-requirement-claude.wav` を再生
> "Claude Codeで要求を追加します。新しい要求を追加してくださいと入力すると、AIが自動的にadd requirementツールを使用します。"

**操作**:
1. `Ctrl + Shift + P` でコマンドパレットを開く
2. "Claude" と入力して Claude Code を起動
3. 以下のテキストを入力:
   ```
   新しい要求を追加してください。タイトルは「ユーザー認証機能」
   ```
4. AI が応答するまで待つ（約12秒）

**コピペ用テキスト**:
```
新しい要求を追加してください。タイトルは「ユーザー認証機能」
```

**待機時間**: 20秒

---

### 📍 シーン5: アウトロ（8秒）

**音声**: `outro.wav` を再生
> "requirements-mcp-serverを使えば、AIと対話しながら効率的に要求管理ができます。"

**操作**:
- 特になし（画面を表示するだけ）

**待機時間**: 8秒

## 録画のコツ

1. **音声との同期**
   - 音声を再生してから操作を開始
   - 慌てず、音声のペースに合わせる
   - 音声が長すぎる場合は、動画編集で調整可能

2. **操作ミスへの対応**
   - 小さなミスは気にせず続行
   - 大きなミスは録画を止めて最初からやり直し
   - シーンごとに分割録画して後で結合も可能

3. **タイミング**
   - 各シーンの待機時間は目安
   - 実際の処理時間に合わせて調整OK

4. **画面レイアウト**
   - ターミナルとClaude Codeの両方が見えるように
   - フォントサイズは大きめ（視聴者が読みやすいように）

---

## トラブルシューティング

### 音声と操作がずれる
→ 後で動画編集で調整可能（おすすめ）

### MCPサーバーの起動が遅い
→ 音声を一時停止してサーバー起動を待つ

### Claude Codeの応答が遅い
→ 待機時間を長めに取る、または動画編集で短縮

---

## 録画後の編集（オプション）

無料の動画編集ソフトを使って品質向上：

- **DaVinci Resolve**（無料版）
  - タイムライン編集
  - 音声と映像の微調整
  - テキスト追加

- **Shotcut**（完全無料）
  - シンプルな編集
  - カット・トリミング
  - フェードイン/アウト

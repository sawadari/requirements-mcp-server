# OBS Studio 録画操作ガイド

## 基本操作

### 🎥 録画の開始・停止

**方法1: ボタンをクリック**
- 右下の **「録画開始」** ボタンをクリック
- 録画中は **「録画終了」** ボタンに変わる
- もう一度クリックで録画停止

**方法2: ショートカットキー（推奨）**
- 録画開始/停止: デフォルトは設定されていない
- 後述の「ショートカットキー設定」で設定推奨

---

## 初回セットアップ

### 1. 録画設定

**ファイル → 設定 → 出力**

1. **出力モード**: 「詳細」を選択
2. **録画タブ**:
   - **録画フォーマット**: `mp4` または `mkv`
   - **エンコーダ**:
     - NVIDIA GPU: `NVIDIA NVENC H.264`
     - AMD GPU: `AMD HW H.264`
     - CPU: `x264`
   - **レート制御**: `CBR`
   - **ビットレート**: `5000 Kbps`（高品質）または `2500 Kbps`（標準）

3. **録画ファイルのパス**:
   - デフォルト: `C:\Users\<ユーザー名>\Videos`
   - 変更推奨: 任意のフォルダを指定

**ファイル → 設定 → 映像**

- **基本解像度**: `1920x1080`（フルHD）
- **出力解像度**: `1920x1080`
- **FPS**: `30` または `60`

**ファイル → 設定 → 音声**

- **デスクトップ音声**:
  - システム音声を録音する場合: デバイスを選択
  - 録音しない場合: `無効`
- **マイク音声**:
  - ナレーションを録音する場合: マイクを選択
  - 録音しない場合: `無効`

---

### 2. ソースの設定（画面キャプチャ）

**ソースパネル（下部中央）**

1. **「+」ボタン** をクリック
2. **「画面キャプチャ」** を選択
3. 名前を入力（例: `デスクトップ`）→ OK
4. キャプチャするモニターを選択 → OK

**VSCodeウィンドウのみ録画する場合**:
1. **「+」ボタン** → **「ウィンドウキャプチャ」**
2. ウィンドウを選択: `[Code.exe]: Visual Studio Code`
3. OK

---

### 3. ショートカットキー設定（推奨）

**ファイル → 設定 → ホットキー**

推奨設定:
- **録画開始/停止**: `Ctrl + F9` または `F10`
- **配信開始/停止**: （使わない場合は空白）

設定後 **「適用」** → **「OK」**

## デモ動画録画の手順

### 準備

1. **OBS Studio を起動**
2. **ソースを確認**:
   - 画面キャプチャまたはウィンドウキャプチャが設定済みか確認
   - プレビュー画面に録画したい内容が表示されているか確認

3. **VSCode を起動**:
   - デモの準備を完了させる
   - ターミナル、Claude Codeなどを開いておく

4. **音声プレーヤーを準備**:
   - Windows Media Player または VLC を起動
   - 最初の音声ファイル `intro.wav` を開く
   - 画面外（録画されない場所）に配置

### 録画開始

1. **OBS で 「録画開始」** をクリック（または `Ctrl + F9`）
2. **音声を再生**: `intro.wav`
3. **デモ操作を実施**: `MANUAL_RECORDING_GUIDE.md` に従う
4. **各シーンで音声を順次再生**

### 録画停止

1. 最後のシーン（outro）が終わったら
2. **「録画終了」** をクリック（または `Ctrl + F9`）
3. 録画ファイルが保存される

## トラブルシューティング

### ❌ 録画ボタンがグレーアウトしている

**原因**: ソースが設定されていない

**解決**:
1. ソースパネルで「+」をクリック
2. 「画面キャプチャ」または「ウィンドウキャプチャ」を追加

### ❌ 録画ファイルが見つからない

**確認方法**:
1. **ファイル → 録画を表示**
2. エクスプローラーで録画フォルダが開く

**または**:
1. **設定 → 出力 → 録画**
2. 「録画ファイルのパス」を確認

### ❌ 音声が録音されていない

**確認**:
1. **設定 → 音声**
2. **デスクトップ音声**: システム音を録音する場合は有効化
3. **マイク音声**: ナレーションを録音する場合は有効化

**音声ミキサー（OBS下部）**:
- 各音声ソースのスライダーを確認
- ミュートになっていないか確認

### ❌ 録画ファイルのサイズが大きすぎる

**対策**:
1. **設定 → 出力 → 録画**
2. **ビットレート** を下げる（例: 5000 → 2500 Kbps）

### ❌ 録画がカクカクする

**対策1**: エンコーダを変更
- **設定 → 出力 → 録画 → エンコーダ**
- GPU エンコーダ（NVENC、AMD HW）を使用

**対策2**: 解像度・FPSを下げる
- **設定 → 映像**
- 出力解像度: `1280x720`
- FPS: `30`

**対策3**: プレビューを無効化
- プレビュー画面を右クリック → **「プレビューを有効化」のチェックを外す**
- 録画自体には影響しない

## 便利な機能

### リプレイバッファ

録画をやり直したい場合に便利:

1. **設定 → 出力 → リプレイバッファ**
2. **「リプレイバッファを有効にする」** にチェック
3. OBS に **「リプレイバッファ開始」** ボタンが表示される
4. 録画ミスした場合、最近の数十秒を遡って保存可能

### シーン切り替え

複数のレイアウトを切り替える場合:

1. **シーンパネル（左下）** で **「+」** をクリック
2. シーン名を入力（例: `オープニング`, `デモ画面`, `エンディング`）
3. 各シーンで異なるソースを設定
4. 録画中にシーンを切り替え可能
